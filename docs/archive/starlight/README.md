# Archived Starlight Reference

This folder preserves high-value Godot Starlight artifacts for recreation and
migration reference. These files are legacy evidence, not Unreal authority.

Contents:

- `data/`: exact Starlight JSON fixtures for Sol, Alpha Centauri, faction
  styles, ship prompts, and the station layout sample.
- `source-scripts/`: copied Godot runtime scripts.
- `tests/`: copied Godot C# tests.
- `scenes-game/`: copied Godot game scene fixtures.
- `roadmap/`: selected Starlight design and balancing docs.
- `roadmap/station_interiors/`: selected station interior design docs.
- `reference/pipelines/`: selected Starlight content pipeline docs.
- `station-design-docs/`: external station blockout and docking-module notes.
- `design/`: selected faction and ship visual reference docs.

Use `../../starlight-recreation-reference.md` for the curated summary and this
archive for exact source fixtures.
