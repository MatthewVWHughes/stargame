# Copy-Paste Ship Prompts

Pipeline:

1. Use the **OpenArt concept prompt** first. This is for finding a design you like.
2. Use the selected image as reference in Tripo/Meshy.
3. Use the **3D prompt** with the reference image. This keeps the generated model structural and game usable.

For Tripo/Meshy triangle output, use the target triangle count listed with each ship.

If a tool has no negative prompt field, paste the `Avoid:` line at the end.
